# UNLOK Landing

Лендинг для проекта UNLOK (Telegram-приложение).

В этом репозитории — основной файл лендинга (LandingPage.tsx).
Полный проект под NDA, показываю только свою часть работы.

## Стек
- React + TypeScript
- Tailwind CSS
- Framer Motion

## Функционал
- Mouse parallax эффекты
- Scroll-based анимации
- Анимированные счётчики
- Кастомные компоненты (Sparkles, FloatingOrbs)
- Адаптивная вёрстка
